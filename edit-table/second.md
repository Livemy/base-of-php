2. 取消添加功能
3. 确认添加按钮，收集UI的数据。=====》 发给data.php
4. data.php收集数据，然后利用sql写入
5. 将新添加的数据进行显示（正常行）

### jQuery中的方法
1. `.html`:
```
html() 方法返回或设置被选元素的内容 (inner HTML)。
如果该方法未设置参数，则返回被选元素的当前内容。
```
2. `val()`:
```
val() 方法返回或设置被选元素的值。

元素的值是通过 value 属性设置的。该方法大多用于 input 元素。

如果该方法未设置参数，则返回被选元素的当前值。
```
